Mise en place de la Base de Données

Prérequis

  • Avoir un serveur MySQL installé
  • Posséder un compte avec les droits de création sur ce serveur

Installation

Un script de mise en place de la BD est fourni, src/bd/install_bd.sh. Pour l'utiliser, il suffit de procéder comme ceci:

  • Se positionner dans le dossier src/bd/
  • chmod +x install_db.sh
  • ./install_bd.sh IP_DU_SERVER_MYSQL USER_MYSQL
  • Entrez le mot de passe du compte MySQL
  • Enjoy
